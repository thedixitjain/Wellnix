Quick Start

1. Install the SDK

pip install wellnix

2. Analyze Nutrition

import wellnix

# Initialize client
client = wellnix.Client(api_key="your-api-key")

# Analyze a nutrition label image
result = client.nutrition.analyze(
    image="path/to/label.jpg",
    user_profile={
        "age": 30,
        "goal": "lose_weight",
        "allergies": ["gluten"]
    }
)

print(result.score)  # 7.5
print(result.explanation)
print(result.recommendations)

3. Analyze Movement

# Analyze exercise form
result = client.muscle.analyze(
    video="path/to/workout.mp4",
    exercise="squat"
)

print(result.form_score)  # 8.2
print(result.reps)  # 12
print(result.feedback)  # ["Keep your knees aligned", ...]

API Endpoints

POST /api/v1/nutri-ai/analyze

Analyze a nutrition label image and get a health score with recommendations.

POST /api/v1/muscle-ai/analyze

Analyze exercise form from video and get form score with corrections.

GET /api/v1/user/scans

Retrieve user's nutrition scan history.

GET /api/v1/user/workouts

Retrieve user's workout session history.
